{% trans "About Publics"%}

{% blocktrans trimmed %} This website was created in the hope that it may be useful to anyone who wants to have better access to the portuguese public information. {% endblocktrans %}

{% blocktrans trimmed %} Our objectives are: {% endblocktrans %}

  • {% blocktrans trimmed %} Help companies to have better access to public information to take more informed decisions. {% endblocktrans %}
  • {% blocktrans trimmed %} Help citizens to have better access to public information and to better understand portuguese public procurements. {% endblocktrans %}
  • {% blocktrans trimmed %} Help scientists and policy makers with a tool to study portuguese open data. {% endblocktrans %}
