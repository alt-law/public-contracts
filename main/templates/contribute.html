{% trans "Why contribute to publicos.pt?" %}

{% blocktrans trimmed %} More than 1.000 persons visit us every day searching for portuguese contracts, companies and laws. This is our way of helping the country: we build tools that help citizens and companies taking better decisions by having better access to public information. {% endblocktrans %}

{% blocktrans trimmed %} We develop such tools because we believe that public information should be as accessible as possible for citizens and companies: we believe that lack of accessibility and transparency decreases economical output, increases bureaucracy and undermines democracy. {% endblocktrans %}

{% blocktrans trimmed %} By contributing to publicos.pt you are thus helping real people to have better access to public information with a hands on approach to help the country. {% endblocktrans %}

{% blocktrans trimmed %} But by contributing to publicos.pt you are also improving your programmers skills: we use state-of-the-art technology and best practices from web development industry and we are always open to help you in any difficulty you may have. {% endblocktrans %}

{% blocktrans trimmed %} As contributor you are expected to execute mini projects where you implement a bug fix or a feature. This provides you valuable experience in designing solutions that don't break existing functionality. {% endblocktrans %}

{% trans "What to contribute?" %}

{% blocktrans trimmed %} A contribution typically falls in one of the following categories: {% endblocktrans %}

  1. {% trans "Error reports or feature suggestions" %}
  2. {% trans "Error fixes or implementation of features" %}

{% blocktrans trimmed %} The first category typically does not require programming skills: navigate on the website and report what you believe should be improved. Any suggestion is welcome! {% endblocktrans %}

{% blocktrans trimmed %} The second category typically requires programming skills: typically Django and Git. If you don't know them but you would still like to help, feel free to ask us for help. {% endblocktrans %}

{% trans "How to contribute?" %}

{% blocktrans trimmed %} As a user of publicos.pt with a suggestion, the easiest way to contribute is to send us an e-mail with it. {% endblocktrans %}

{% blocktrans trimmed %} As a developer, you can either pick an existing issue in our bug tracker, or open a new one in Github. Check the details on our page for developers. {% endblocktrans %}

{% blocktrans trimmed %} Like most websites, publicos.pt uses continuous deployment: once your contribution is pushed to the repository's master branch, it is deployed on the server and your contribution is used by all users of the website. {% endblocktrans %}
