Bem vindo ao Públicos

Conheça os contratos, as leis e os deputados do estado português.

Este website combina informação pública de diferentes fontes oficiais e unifica-a. Em particular, estamos interessados em:

  • Contratos
  • Leis
  • Deputados

Contratos

Neste website pode encontrar todas as entidades, concursos públicos e contratos públicos presentes na base de dados oficial.

  • Pode conhecer que empresas são mais contratadas, que empresas contractam e em que sector económico.
  • Pode conhecer os últimos concursos públicos, ou que entidades públicas estão a contratar mais.
  • Pode estar actualizado das últimas novidades sem ter de nos visitar: temos RSS de todas as listas.

Para mais informações, consulte esta página.

Leis

Este website tem todas as leis presentes no site oficial do Diário da República.

  • [Em construção] Pode procurar no texto integral da lei.
  • [Em construção] Temos RSSs das listas de leis, caso queira receber actualizações no seu computador quando novas leis surgem.

Para mais informações, consulte esta página.

Deputados

Este website tem a lista de todos os deputados presente no site do parlamento português.

  • Divulgamos a lista de todos os deputados que passaram pelo parlamento.
  • [Em construção]: temos os registos de interesses de todos os deputados, incluindo ligações para as entidades presentes em contratos públicos.
  • [Em construção]: Temos os registo de que leis os deputados proposeram, ligando com as leis já existentes.

Para mais informações, consulte esta página.

Outras informações:

  • Este website é open source, público e não-governamental.
  • Todos os dados aqui apresentados estão sincronizados com os dados oficiais com o máximo de 24h de erro.
  • Toda a base de dados presente neste website é pública. Consulte a documentação do website para a aceder.
