Código de conduta

A maioria das pessoas não tem o tempo, vontade ou conhecimentos para escrutinar as análises que são feitas neste website, ainda que o código seja aberto. Para além disso, este trabalho não é peer reviewed e portanto não é científico. Seja crítico na interpretação.

As análises que aqui apresentamos podem ser facilmente usadas para difamar entidades específicas, quer sejam públicas ou privadas. Em qualquer circunstância este projeto se revê nessas afirmações. Este é um projeto para conhecer e compreender a contratação portuguesa e não um local para formar uma agenda política.

Se tiver alguma crítica ou sugestão, pode colocá-la na mailing list pública deste projeto, de forma a ela ser publicamente discutida.
