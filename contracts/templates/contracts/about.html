Sobre este projeto

Este projeto está sobre a responsabilidade de Jorge C. Leitão.

As análises neste site não constituem um trabalho científico:

Este trabalho não é peer reviewed. Consequentemente, é necessário ser crítico na interpretação dos resultados apresentados.

Fazemos um esforço sério para que os resultados sejam rigorosos e interessantes, mas não há garantias que não possam existir erros.

Se tiver alguma crítica ao trabalho apresentado, pode colocá-la na mailing list pública deste projeto, de forma a ela ser discutida publicamente.
