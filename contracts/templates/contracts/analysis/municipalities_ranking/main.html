Ranking de transparência contratual dos municípios portugueses

Por lei, os municípios portugueses têm de publicar os seus contratos numa base de dados oficial. Como é que o município publica os seus contratos é uma forma de quantificar a eficiência dos seus processos administrativos.

O publicos.pt tem analisado esta base de dados e nesta página sumariza alguns dos indicadores que tem desenvolvido para comparar a transparência contratual dos municípios portugueses.

Base de dados

A base de dados usada é a base de dados oficial dos contratos públicos, BASE. Como a base de dados não diferencia municipios de não municipios, os NIFs desta base de dados foram cruzados com os da base de dados de entidades públicas (que diferencia municípios) do Banco de Portugal (ficheiro excel).

Todos os contratos de cada município em cada ano foram selecionados e alguns indicadores de performance do município foram calculados. Os municípios foram depois ordenados (ranking) de acordo com o seu performance em cada indicador individual.

Indicadores:

Tempo médio de publicação:

É o valor médio de (data de inserção - data de assinatura), onde "data de inserção" é a data em que o contrato foi adicionado à base de dados oficial e "data de assinatura" é a data em que o contrato foi assinado.

O tempo médio de publicação é assim um indicador de eficiência administrativa do município.

Especificidade média

A União Europeia tem um sistema de categorização de contratos públicos (CPVS), que define ~9000 categorias de acordo com uma hierarquia de ~7 níveis de especificidade (e.g. categoria "fruta" está dentro da categoria "produtos alimentares").

A especificidade de um contrato é o nível (0-7) na hierarquia da categoria do contrato quando este é adicionado. Há contratos em que o município não escolheu qualquer categoria (nível 0), e outros em que escolheu uma categoria específica (>4).

A especificidade média é assim um indicador de transparência do município pois indica quanto esforço o município faz para os seus contratos serem adicionados correctamente.

Percentagem de publicação correcta

Encontrar contratos públicos numa base de dados tipicamente requer uma busca textual (e.g. google). Para isso, é importante que quando o contrato é adicionado, contenha uma descrição do mesmo. Publicações correctas são aquelas que contém essa descrição.

Este indicador é forma de quantificar a transparência do município.
