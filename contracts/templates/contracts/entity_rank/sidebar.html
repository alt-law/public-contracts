Pontos chave

  • A base de dados de contratos públicos é uma ferramenta útil para analizar entidades públicas.
  • Usámos um indicador, consistente com as recomendações oficiais, para comparar a especificidade média que diferentes municípios usa na divulgação dos seus contratos em domínio público.
  • Como mais especificidade leva a maior transparência mas requer trabalho, podemos comparar quantitativamente o esforço que diferentes municípios fazem para serem transparentes na divulgação dos seus contratos.
