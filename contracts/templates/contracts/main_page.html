Analizar e visualizar a rede de contratos públicos portugueses

Os contratos de entidades públicas portuguesas estão publicados numa base de dados oficial. Esta base de dados não permite fazer buscas complexas para análises estatísticas (por uma boa razão!).

Neste projeto, desenvolvemos uma forma simples de qualquer pessoa analisar a base de dados. Como? Mantemos uma base de dados sincronizada regularmente com base de dados oficial e temos um código open source e em Python, que permite fazer qualquer pergunta à base de dados com o mínimo de conhecimentos de Python.

A ideia é diminuir a barreira técnica que existe entre o interesse comum e a informação já existente.

Este website, também open source, é o local onde divulgamos resultados de análises que nós próprios fazemos com o código que disponibilizamos, e tem também uma interface gráfica que lhe permite navegar na complexa rede de contratos públicos.

O gráfico é um exemplo do que estamos a fazer: representa o número relativo de contratos (eixo y) com um dado valor em euros (eixo x). Neste exemplo, toda a base de dados existente - ~370000 contratos - foi considerada.

Explique-mos o gráfico: considere o valor 10^6 (1 milhão de euros) no eixo do x. Nesse ponto, o valor correspondente no eixo dos y é de ~10^-1 = 0.1 = 10%. Isto significa que 1 em cada 10 contratos públicos são de 1 milhão de euros. Por outro lado, só 1 em cada 10^4 (10 mil, eixo y) contratos são de 10^9 (mil milhões de euros, eixo x).

Uma análise mais cuidada no gráfico mostra que:

  • a distribuição tem um máximo centrado em ~10^6 = 1.000.000 (1 milhão de euros).
  • Entre ~10^7 e ~10^9 a distribuição tem um comportamento lei de potências (linha a vermelho é expoente -1).
