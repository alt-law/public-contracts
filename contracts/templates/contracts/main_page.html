Analizar e visualizar a rede de contratos públicos portugueses

Os contratos de entidades públicas portuguesas estão publicados numa base de dados oficial. Esta base de dados não permite fazer buscas complexas para análises estatísticas (por uma boa razão!).

Neste projeto, mantemos uma base de dados sincronizada regularmente com a base de dados oficial e temos um código open source e em Python, que permite fazer análises estatísiticas à base de dados com mínimo de conhecimentos em Python.

A ideia é diminuir a barreira técnica que existe entre o interesse comum e a informação já existente.

Este website, também open source, é o local onde divulgamos resultados de análises que nós próprios fazemos com o código que disponibilizamos, e permite navegar na complexa rede de contratos públicos.
