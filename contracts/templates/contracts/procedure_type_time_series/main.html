Evolução temporal da contratação pública portuguesa - concurso público ou ajuste direto?

{% url "law_view" 299134 as public_contracts %} A contratação pública em Portugal é feita maioritariamente por

  • ajuste direto,
  • concurso público,
  • concurso limitado por prévia qualificação ou
  • acordo quadro.

Do ponto de vista legislativo é relevante saber quanto cada um é usado pois permite uma análise quantitativa de quais os tipos relevantes na lei.

Assim, neste trabalho respondemos à pergunta: qual a percentagem de cada um destes tipos?

Considerámos todos os contratos da base de dados pública fechados em cada mês desde 2010 e cálculamos as percentagens feitas de cada um dos tipos.

{% url 'contracts_data_selector' 'procedure-types-time-series-json' as the_url %} {% include "contracts/procedure_type_time_series/graph.html" with url=the_url%}

De 2010 a 2013, dentro dos contratos existentes na base de dados, houve um aumento consistente para o uso de concurso público (de ~1% para ~10%) que se tem invertido em 2014.

Os acordos-quadro, uma forma de contratação que só em 2014 foi adicionada na base de dados oficial, que explica o seu início nessa altura.

Dados deste mês ainda não são conclusivos

O tempo médio entre o contrato ter sido fechado e a publicação do mesmo na base de dados oficial é por volta de 1-2 meses.

Assim, os últimos dois meses ainda podem sofrer alterações e portanto devemos esperar até os valores estabilizarem para tirarmos conclusões.

Ajustes directos dominam

Como já reportado pelo Instituto de Construção e Imobiliário I.P. para 2012 (relatório oficial, Quadro 9), os ajustes directos dominam a contratação pública portuguesa, embora em 2014 tenham atingido pela primeira vez menos de 90% de toda a contratação reportada.
