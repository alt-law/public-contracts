{% blocktrans %} The types of laws that ever exist or existed in the Portuguese Law are shown since the first Republic. {% endblocktrans %}

{% blocktrans %} In this work I take a look at how the number of different laws evolved over time. To that, I considered all documents in the Series I of "Diário da República", and plotted the time evolution of the number of documents of a given type. {% endblocktrans %}

{% trans "How to use" %}

{% blocktrans %} To exclude one type, click on it; to select only one type, double click on it. Over the mouse for detailed information on each year. {% endblocktrans %}

{% trans "Details" %}

{% blocktrans %} Only the 17 most common types are shown to simplify the visualization; "Outros" represent the others types; types are ordered by commonness ("Portaria" is the commonest, then "Decreto", etc.) {% endblocktrans %}

{% blocktrans %} Here we are interested in the phenomena of creation of new laws; naturally most of the documents here were already revoked. {% endblocktrans %}

{% trans "Observations" %}

  • {% blocktrans %} Portaria is by far the most common type of document; it was used consistently throughout our modern history. {% endblocktrans %} {% blocktrans %} This is consistent since portarias are administrative and regulatory laws with narrow scope. {% endblocktrans %}

    {% blocktrans %} It is relevant to point out that since 2011 it has been used very little when compared to previous decades; only in the 50' Portaria was used this little. {% endblocktrans %}

  • {% blocktrans %} Decreto was used very often until the dictatorship (1933), and it seems to have been replaced by other types, notably Declaração and Decreto-Lei. {% endblocktrans %}

  • {% url "law_view" 166072 as the_url %} {% blocktrans %} If we exclude "Portaria", we observe that, overall, the number of laws is decreasing since 1976 (year of approval of a new Constitution). {% endblocktrans %}

  • {% blocktrans %} In radical governmental changes (1933, 1974) new types appear and old become extinct [hide "Portaria" and "Decreto" to notice this] {% endblocktrans %}

    {% blocktrans %} "Decreto-Lei" is highly used immediately after these changes, which is consistent with its used to define general principles [hide "Portaria" and "Decreto" to see this better]. {% endblocktrans %}

  • {% blocktrans %} Declaração seems to have been used less in recent years, but in fact it has been replaced by more specific types of declarations (also part of "Others"). {% endblocktrans %}

  • {% blocktrans %} Further observations are left as exercise to the reader. ;) {% endblocktrans %}
